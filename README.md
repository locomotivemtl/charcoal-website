## Requirements

| Name        | Version |
| ----------- | ------- |
| [PHP]       | > 7.2   |

| [Charcoal]  | > 2021  |

| Name       | Version |
| ---------- | ------- |
| [Node]     | > 14.15 |

## Installation

```sh
git clone https://github.com/locomotivemtl/charcoal-website
cd charcoal-website
composer install
npm install
```

## Usage

```sh
# watch
npm start

# minify
npm run build
```

See [build.js](blob/master/build/build.js) and [watch.js](blob/master/build/build.js) for detaild.
